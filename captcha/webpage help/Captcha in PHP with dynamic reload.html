Captcha in PHP with dynamic reload
Captcha is used to avoid Spam users in websites. It is also used to avoid spam comments in our blog or websites. 
How captcha will filter spam users? 
Suppose you have a form with the fields of username, email, message with the action of POST with the action of `action.php` file. Any user can post values or spam values without using your website or your form page in browser. They can simply POST the values to your site.com/action.php. To avoid this, we shall use captcha to authenticate that the user is human. Here is the simple captcha ready to integrate in your website which contains dynamic captcha reload without page reload.
  <?php
session_start();
$msg = '';
if(isset($_POST["submit"])){
    $original_captcha = $_SESSION["captcha"]; // session 'captcha' has been alreadey created in captcha.php file, if you want to rename the session open the captcha.php file and find and change the name of the session
    $user_captcha = $_POST["captcha"];
    if($user_captcha == $original_captcha){
        $msg = '<span class="success">Captcha Matched!</span>';
    }else{
        $msg = '<span class="error">Oops! Captcha Mismatched!</span>';
    }
}
?>
